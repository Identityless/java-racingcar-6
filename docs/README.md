## 구현 기능 목록

### 차량 정보를 담을 클래스
    - 차량 이름 필드
    - 차량 기리 저장 필드
    - 차량 전진 메서드
### 경주 차량 정보를 담을 일급 컬렉션
### 사용자 입력 기능
    - 싱글톤으로 관리
    - 자동차 이름 입력 메서드
    - 시도할 횟수 입력 메서드
### 자동차 경주 기능
    - 싱글톤으로 관리
    - 랜덤 숫자 발생 메서드
    - 조건에 부합하는 차량 전진 메서드
### 게임 결과 출력 기능
    - 실행 결과 출력 메서드
    - 최종 우승자 출력 메서드
